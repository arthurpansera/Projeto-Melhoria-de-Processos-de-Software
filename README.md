# Projeto Melhoria de Processos de Software

Projeto acadêmico voltado à avaliação e melhoria de processos de desenvolvimento de software em uma empresa real, utilizando como referência o modelo de qualidade MPS.BR e técnicas de modelagem de processos de negócio.
